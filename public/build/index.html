Environment Setup

  1. Download the Project Starter Files.
  2. Double-click the zipped file to unzip it.
  3. Move the unzipped folder to your Desktop.
  4. (Mac) Drag your project folder onto the Sublime Text icon in the dock. (PC) Open SublimeText and open the project from the Menu.

Programming Workflow

  1. View the Game in Chrome:
    • Double-click the index.html file to open in a Google Chrome browser.
    • You will see the game titlescreen.
  2. Test Code Changes Often:
    • Save the code you typed.
    • Switch to your Google Chrome browser that contains your game.
    • Refresh the page by hitting Command-R
  3. Debugging:
    • In Chrome, right-click (Mac: Control-Click) the game page and choose Inspect from the dropdown.
    • Click Console in the nav. If there any errors in your code if will show along with hints to the line number of the error.
